# 🚀 AI Developer Cheerleader

A simple Python application built during **Global Hack Week: Cloud** to solve the problem of developer burnout and fatigue.

## 📝 Description
I built this simple Python app with the help of **GitHub Copilot** to stay motivated during GHW Cloud. It randomly selects a motivational quote to keep developers going during long nights of coding! 

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **AI Tool:** GitHub Copilot (used for logic generation and rapid prototyping)
* **Environment:** GitHub Codespaces

## ⚡ How it Works
The script uses the `random` library to pull from a curated list of encouraging quotes and prints them to the console in a clean, readable format.

## 🧠 What I Learned
* How to use Copilot's inline chat to generate Python functions.
* Managing repository workflows during a high-pressure hackathon.
